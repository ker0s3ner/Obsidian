## Definition

**Ordinary Differential Equation (ODE)**: Equation involving unknown function $y(x)$ and its derivatives.

General form:
$$F(x, y, y', y'', \ldots, y^{(n)}) = 0$$

**Order**: Highest derivative appearing in equation.

**First-order**: Only involves $y'$ (first derivative).

---

## Standard Forms

### Explicit Form

$$\frac{dy}{dx} = f(x, y)$$

or using prime notation:
$$y' = f(x, y)$$

### Implicit Form

$$F(x, y, y') = 0$$

### Initial Value Problem (IVP)

$$y' = f(x, y), \quad y(x_0) = y_0$$

Specifies solution passing through point $(x_0, y_0)$.

---

## Separable Equations

### Definition

Equation is **separable** if can write:
$$\frac{dy}{dx} = g(x)h(y)$$

### Solution Method

1. Separate variables:
   $$\frac{dy}{h(y)} = g(x)\,dx$$

2. Integrate both sides:
   $$\int \frac{dy}{h(y)} = \int g(x)\,dx$$

3. Solve for $y$ (if possible)

```graph2d
y' = x*y
```

> [!example] Visual Interpretation
> Slope field for $y' = xy$. Solutions are exponential curves.

### Example

Solve $y' = xy$ with $y(0) = 1$:

Separate: $\frac{dy}{y} = x\,dx$

Integrate: $\ln|y| = \frac{x^2}{2} + C$

Solve: $y = Ae^{x^2/2}$ where $A = e^C$

Initial condition: $1 = A$, so $y = e^{x^2/2}$

## Linear First-Order Equations

### Standard Form

$$\frac{dy}{dx} + P(x)y = Q(x)$$

Linear in $y$ and $y'$.

### Integrating Factor Method

1. Compute integrating factor:
   $$\mu(x) = e^{\int P(x)\,dx}$$

2. Multiply equation by $\mu(x)$:
   $$\mu(x)y' + \mu(x)P(x)y = \mu(x)Q(x)$$

3. Left side becomes derivative:
   $$\frac{d}{dx}[\mu(x)y] = \mu(x)Q(x)$$

4. Integrate:
   $$\mu(x)y = \int \mu(x)Q(x)\,dx + C$$

5. Solve for $y$:
   $$y = \frac{1}{\mu(x)}\left[\int \mu(x)Q(x)\,dx + C\right]$$

### Example

Solve $y' + 2xy = x$:

$P(x) = 2x$, so $\mu(x) = e^{\int 2x\,dx} = e^{x^2}$

Multiply: $e^{x^2}y' + 2xe^{x^2}y = xe^{x^2}$

Recognize: $\frac{d}{dx}[e^{x^2}y] = xe^{x^2}$

Integrate: $e^{x^2}y = \int xe^{x^2}\,dx = \frac{1}{2}e^{x^2} + C$

Solve: $y = \frac{1}{2} + Ce^{-x^2}$

## Exact Equations

### Definition

Equation $M(x,y)\,dx + N(x,y)\,dy = 0$ is **exact** if:
$$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$$

### Solution Method

If exact, exists potential function $\psi(x,y)$ such that:
$$\frac{\partial \psi}{\partial x} = M, \quad \frac{\partial \psi}{\partial y} = N$$

Solution: $\psi(x,y) = C$

Find $\psi$ by:
1. Integrate $M$ with respect to $x$: $\psi = \int M\,dx + h(y)$
2. Differentiate with respect to $y$ and match with $N$
3. Solve for $h(y)$

### Example

Solve $(2xy + y^2)\,dx + (x^2 + 2xy)\,dy = 0$:

Check: $\frac{\partial M}{\partial y} = 2x + 2y = \frac{\partial N}{\partial x}$ ✓ Exact!

Integrate $M$: $\psi = \int (2xy + y^2)\,dx = x^2y + xy^2 + h(y)$

Differentiate: $\frac{\partial \psi}{\partial y} = x^2 + 2xy + h'(y) = N = x^2 + 2xy$

So $h'(y) = 0$, thus $h(y) = C$

Solution: $x^2y + xy^2 = C$

## Homogeneous Equations

### Definition

Equation $y' = f(x,y)$ is **homogeneous** if $f(tx, ty) = f(x,y)$ for all $t$.

Equivalently: $f(x,y) = F(y/x)$.

### Solution Method

Substitute $v = y/x$, so $y = vx$ and $y' = v + xv'$.

Equation becomes separable in $v$ and $x$.

### Example

Solve $y' = \frac{y}{x} + \frac{x}{y}$:

Let $v = y/x$, then $y' = v + xv'$:
$$v + xv' = v + \frac{1}{v}$$

Simplify: $xv' = \frac{1}{v}$

Separate: $v\,dv = \frac{dx}{x}$

Integrate: $\frac{v^2}{2} = \ln|x| + C$

Back-substitute: $\frac{y^2}{2x^2} = \ln|x| + C$

---

## Bernoulli Equations

### Standard Form

$$y' + P(x)y = Q(x)y^n$$

where $n \neq 0, 1$.

### Solution Method

Substitute $v = y^{1-n}$, which transforms to linear equation.

Derivative: $v' = (1-n)y^{-n}y'$

After substitution, get linear equation in $v$.

### Example

Solve $y' + y = xy^2$:

This is Bernoulli with $n=2$.

Let $v = y^{-1}$, so $v' = -y^{-2}y'$.

Multiply original by $-y^{-2}$:
$$-y^{-2}y' - y^{-1} = -x$$

Substitute: $v' - v = -x$

Linear equation! Solve with integrating factor $\mu = e^{-x}$.

## Common Mistakes

> [!warning] Forgetting Constant of Integration
> Always include $+C$ when integrating. Apply initial conditions last.

> [!warning] Dividing by Zero
> When separating variables, check if $h(y) = 0$ gives additional solutions.

> [!warning] Confusing Linear and Nonlinear
> Linear: $y'$ and $y$ appear to first power only. Nonlinear: $y^2$, $\sin(y)$, etc.

6. Find the particular solution of $y' = 2xy$ satisfying $y(1) = e$

8. Determine whether $y' = x^2 + y^2$ is separable, linear, or neither
